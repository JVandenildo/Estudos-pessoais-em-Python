## Bundle of small Python tips

### TEsting code snippets

Two lines `# %%` may be used to run code snippet as a cell from Jupyter Notebook to test that code snippet. Example:

```python
# %%
def sum(a, b):
    return a+b

print(sum(2,3))
# %%
```
