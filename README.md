# Studies In Python

Repository for Python algorithm/syntax testing.

## Conjunto de dicas para Python

### Testando trechos de código

Usar duas linhas `# %%` testa o que estiver entre elas como se fosse uma célula de Jupyter. Exemplo:

```python
# %%
def sum(a, b):
    return a + b

print(sum(2, 3))
# %%
```
